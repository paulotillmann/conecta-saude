# Design System do Projeto Conecta Saúde

Este projeto utiliza o arquivo de design do Conecta Saúde como a única fonte da verdade para o design system.

- **Design System File**: [design-conectasaude.md](file:///C:/CONECTA-Saude/design-conectasaude.md)

## Diretrizes de UI/UX
Toda interface de usuário, componentes de UI, esquemas de cores, espaçamentos, tipografia e estilos devem seguir estritamente as especificações contidas no arquivo [design-conectasaude.md](file:///C:/CONECTA-Saude/design-conectasaude.md).

- **Cores principais**: Burgundy institucional (`#601010` / `#3f0003`) como Primary, cinza escuro (`#5e5e5e`) como Secondary, e fundo claro (`#fff8f7`) como Background/Surface.
- **Tipografia**: Utilizar a fonte **Inter** com os pesos e tamanhos recomendados no arquivo de design.
- **Bordas e Containeres**: Bordas arredondadas (radius) de 8px para botões/inputs e 12px para cards.
